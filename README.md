# ETL Mini Project

This repository demonstrates an end‑to‑end ETL workflow using **Python +
Pandas**, including: - Data extraction from CSV, JSON, Excel files\
- Data cleaning and standardization\
- Dimension table creation\
- Fact table enrichment\
- Aggregations (yearly profit, customer‑level profit, etc.) - Automated
testing using **pytest**
